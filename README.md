## 🎯 项目核心要点

### 业务模式

```
手机卡设备（提供 IP）→ 公网服务器（中转）→ 用户（使用代理）
```

**价值主张：**
- 用户：获得真实的住宅 IP，绕过限制
- 平台：流量变现，赚取差价
- 设备主：分享闲置流量，获得收益

---

### 技术方案

**核心问题：** NAT 后的设备如何对外提供服务？

**解决方案：** 反向连接 + 中继服务器

```
设备主动连接公网服务器 → 建立反向隧道 → 用户通过服务器访问设备
```

**关键技术：**
- frp（反向代理）
- gost（SOCKS5 服务器）
- Docker（容器化部署）
- FastAPI（API 服务）

---

1. **DDoS 防护：** 使用 Cloudflare 等 CDN
2. **访问限制：** 限制单用户并发连接数
