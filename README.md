# Cell Proxy IP - net终端项目

> 基于手机卡流量的IP代理服务平台  
> RV1103 + EC600M双卡方案 + MQTT调度平台

---

## 📖 项目概述

**一句话介绍：**
> 为客户提供海外手机卡IP代理服务的整套解决方案（硬件 + MQTT调度平台 + 中转服务器）。

**商业模式：** B2B2C
- **我们提供：** net终端硬件、MQTT调度平台、中转服务器系统、技术维护
- **客户（运营方）：** 购买硬件、管理手机卡、对外提供IP代理服务运营
- **部署地：** 菲律宾（前期30-100台测试）

## 🎯 核心功能

### 产品定位
- 海外部署（菲律宾为主）
- 手机卡IP代理服务
- 远程管理和运维

### 技术亮点
1. **双网卡架构** - 手机卡（代理流量）+ RJ45（MQTT/SSH）
2. **双SIM卡自动切换** - SIM1失败自动切换SIM2，保证高可用
3. **MQTT远程管理** - 即使手机卡断网也能通过RJ45远程诊断
4. **OTA固件升级** - 批量远程升级设备固件

### 30天P0功能
- ✅ 双网卡切换
- ✅ MQTT远程管理
- ✅ SIM卡自动切换
- ✅ 远程故障诊断
- ✅ 流量统计
- ✅ 设备运维控制
- ✅ 批量固件升级

## 🏗️ 技术架构

```
┌──────────────────────────────────────────────────┐
│              net终端（RV1103）                    │
│                                                  │
│  网卡1：手机卡（usb0）                            │
│    - 代理流量（核心业务）                         │
│    - MQTT通讯（RJ45不可用时）                     │
│                                                  │
│  网卡2：RJ45（eth0）                              │
│    - MQTT通讯（优先）                             │
│    - SSH远程访问                                  │
│                                                  │
│  双SIM卡：                                        │
│    - SIM1 / SIM2 自动切换                         │
│    - 信号监控和网络注册检测                       │
└──────────────────────────────────────────────────┘
         ↓ MQTT / frpc
┌──────────────────────────────────────────────────┐
│           MQTT调度平台 + 中转服务器               │
│  - 设备管理                                       │
│  - 远程命令                                       │
│  - 流量统计                                       │
│  - SOCKS5代理中转                                 │
└──────────────────────────────────────────────────┘
```
