## 📝 归档规则

### 归档时机

- 报告超过30天
- 文档版本更新后的旧版本
- 被新文档替代的临时文档
- 测试期间的实验性文档

### 归档流程

1. 将旧文档移动到对应的归档子目录
2. 保持原文件名或添加日期前缀
3. 在归档目录的README中记录归档原因
4. 提交Git时注明"归档旧文档"
