## 🏗️ 技术架构

### 设备端（RV1103）
- Python 3.11 - 网络监控、MQTT客户端
- tiny-frpc - 反向隧道
- microsocks - SOCKS5代理

### 服务端
- **Go + PostgreSQL** - MQTT调度服务
- Mosquitto - MQTT Broker
- frps - 反向代理服务器
- Grafana - 监控面板
