## 使用示例

### Curl命令
```bash
# 使用最稳定的端口
curl -x socks5://username:password@codex.adusun.com:21883 https://ip.sb

# 检查出口IP
curl -x socks5://username:password@codex.adusun.com:21885 https://ipinfo.io
```

### Python requests
```python
import requests

proxies = {
    'http': 'socks5://username:password@codex.adusun.com:21883',
    'https': 'socks5://username:password@codex.adusun.com:21883'
}

response = requests.get('https://ip.sb', proxies=proxies)
print(response.text)
```
