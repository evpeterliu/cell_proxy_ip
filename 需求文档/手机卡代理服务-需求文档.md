## 📖 项目概述

### 项目目标

构建一个**基于手机卡流量的 SOCKS5 代理服务平台**，让用户可以：
- 使用真实的手机卡 IP 地址访问互联网
- 获得动态住宅 IP（每次重连 IP 可能变化）
- 绕过某些网站的 IP 限制

### 核心价值

- **对用户：** 提供高质量的住宅 IP 代理服务
- **对平台：** 利用闲置手机卡流量变现
- **对终端设备主：** 分享手机卡流量获得收益

## 🔧 核心问题与解决方案

### 问题 1：NAT 后的设备如何对外提供服务？

**问题描述：**
- 手机卡设备在运营商的 NAT 网络后面
- 没有公网 IP 地址
- 外部用户无法直接连接

**解决方案：反向连接 + 中继服务器**

```
外部用户 → 公网服务器 → NAT 后的手机卡设备
         (中转)      (反向连接)
```

**通俗解释：**
就像你在家里（NAT 后），外面的人无法直接敲你家门。但你可以主动打电话（反向连接）给中介公司（公网服务器），中介公司接到客户后，通过你打来的电话线转接给你。

---

### 问题 2：所有流量都要经过公网服务器吗？

**答案：是的，必须经过**

**原因：**
1. 手机网络是运营商级 NAT（CGNAT），无法配置端口映射
2. 外部用户无法直接连接到 NAT 后的设备
3. 必须通过公网服务器中转所有数据

**能否优化：**
- ✅ 可以尝试 UDP 打洞技术，成功率约 15-20%
- ✅ 成功后可以直连，节省带宽
- ❌ 但 80-85% 的情况仍需中继

---

### 问题 3：如何降低公网服务器成本？

**方案对比：**

| 方案 | 带宽消耗 | 成本 | 说明 |
|------|---------|------|------|
| **国内服务器** | 100% 中转 | ¥30,000/月 | 带宽贵 |
| **国外服务器** | 100% 中转 | ¥1,700/月 | 带宽便宜 |
| **智能混合** | 80% 中转 | ¥1,400/月 | 部分直连 |

**推荐：使用国外服务器（Hetzner/OVH）+ 智能混合方案**

**4. 技术实现要点**

**AT命令清单：**
```bash
# 1. 信号强度查询
AT+CSQ
# 返回：+CSQ: <rssi>,<ber>
# 示例：+CSQ: 25,99  (rssi=25, 信号良好)

# 2. 网络注册状态查询
AT+CREG?
# 返回：+CREG: <n>,<stat>
# stat=1: 已注册本地网络
# stat=5: 已注册漫游网络
# stat=0/2/3: 未注册

# 3. SIM卡ICCID查询
AT+ICCID
# 返回：+ICCID: 89860123456789012345
# 如果无SIM卡，返回ERROR

# 4. 运营商查询
AT+COPS?
# 返回：+COPS: 0,0,"Globe",7
# 0,0: 自动选择
# "Globe": 运营商名称
# 7: 网络类型（LTE）

# 5. SIM卡切换
AT+QDSIM=<sim_id>
# sim_id=0: 切换到SIM1
# sim_id=1: 切换到SIM2

# 6. 断网重新拨号
AT+CGACT=0,1  # 断开数据连接
AT+CGACT=1,1  # 重新建立数据连接

# 7. 飞行模式
AT+CFUN=0     # 进入飞行模式
sleep 5       # 等待5秒
AT+CFUN=1     # 退出飞行模式

# 8. 🆕 查询模块联网状态
AT+QNETDEVCTL?
# 返回：+QNETDEVCTL: <mode>,<status>,<enable>
# mode: 网络设备模式（1-11）
# status: 连接状态（0-3）
# enable: 是否启用（0或1）

# 9. 🆕 启用网络设备并拨号
AT+QNETDEVCTL=3,1,1
# 参数1=3: ECM模式（Ethernet Control Model）
# 参数2=1: 启用/连接
# 参数3=1: 自动拨号
# 说明：此命令打开模块拨号上网功能，模块会自动拨号

# 10. 🆕 配置USB网络模式
AT+QCFG="usbnet",1
# 参数=1: ECM模式
# 参数=0: QMI模式
# 说明：设置后需要重启模块生效

# 11. 🆕 配置APN
AT+CGDCONT=1,"IP","internet"
# 菲律宾运营商APN：
# Globe: AT+CGDCONT=1,"IP","internet.globe.com.ph"
# Smart: AT+CGDCONT=1,"IP","smartlte"

# 12. 🆕 启用自动附着网络
AT+CGATT=1
# 参数=1: 自动附着
# 参数=0: 手动附着

# 13. 🆕 查询IP地址
AT+CGPADDR=1
# 返回：+CGPADDR: 1,"10.123.45.67"
```

**8. 异常处理代码示例**

```python
class SIMCardManager:
    def __init__(self):
        self.current_sim = 1  # 当前使用的SIM卡（1或2）
        self.retry_count = 0
        
    def check_network(self):
        """主流程：检查网络并处理异常"""
        # 步骤1：检查当前SIM卡状态
        if not self.check_sim_signal():
            # 无信号，直接切换
            return self.switch_to_other_sim()
        
        # 步骤2：检查网络注册
        if not self.check_network_registration():
            # 未注册，等待30秒后重试
            time.sleep(30)
            if not self.check_network_registration():
                # 仍未注册，切换SIM卡
                return self.switch_to_other_sim()
        
        # 步骤3：尝试联网
        if not self.try_connect():
            # 联网失败，进入断网重连流程
            return self.reconnect_flow()
        
        return True
    
    def reconnect_flow(self):
        """断网重连流程（最多2次）"""
        # 第1次：断网重新拨号
        if self.disconnect_and_redial():
            return True
        
        # 第2次：飞行模式后再联网
        if self.airplane_mode_reconnect():
            return True
        
        # 2次都失败，切换SIM卡
        self.log_error(f"SIM{self.current_sim}断网重连2次失败")
        return self.switch_to_other_sim()
    
    def switch_to_other_sim(self):
        """切换到另一张SIM卡"""
        other_sim = 2 if self.current_sim == 1 else 1
        
        # 步骤1：检查SIM卡是否插入
        if not self.check_sim_card_exists(other_sim):
            self.log_error(f"SIM{other_sim}未插入")
            time.sleep(60)
            return False
        
        # 步骤2：检查网络注册状态
        if not self.check_sim_registration(other_sim):
            self.log_error(f"SIM{other_sim}未注册到网络")
            time.sleep(60)
            return False
        
        # 步骤3：切换SIM卡
        self.execute_at_command(f"AT+QDSIM={other_sim-1}")
        self.current_sim = other_sim
        
        # 步骤4：尝试联网
        if not self.try_connect():
            # 联网失败，进入断网重连流程
            return self.reconnect_flow()
        
        return True
    
    def check_sim_card_exists(self, sim_id):
        """检查SIM卡是否插入（AT+ICCID）"""
        response = self.execute_at_command("AT+ICCID")
        if "ERROR" in response:
            return False
        if len(response) > 10:  # ICCID通常是19-20位
            return True
        return False
    
    def check_sim_registration(self, sim_id):
        """检查SIM卡网络注册状态（AT+COPS?）"""
        response = self.execute_at_command("AT+COPS?")
        # 检查是否已注册到运营商
        if "Globe" in response or "Smart" in response:
            return True
        
        # 未注册，等待30秒
        time.sleep(30)
        response = self.execute_at_command("AT+COPS?")
        if "Globe" in response or "Smart" in response:
            return True
        
        return False
    
    def disconnect_and_redial(self):
        """断网重新拨号"""
        self.execute_at_command("AT+CGACT=0,1")  # 断开
        time.sleep(2)
        self.execute_at_command("AT+CGACT=1,1")  # 重连
        time.sleep(5)
        return self.check_internet_connection()
    
    def airplane_mode_reconnect(self):
        """飞行模式后再联网"""
        self.execute_at_command("AT+CFUN=0")  # 飞行模式
        time.sleep(5)
        self.execute_at_command("AT+CFUN=1")  # 退出飞行模式
        time.sleep(10)
        return self.check_internet_connection()
```
